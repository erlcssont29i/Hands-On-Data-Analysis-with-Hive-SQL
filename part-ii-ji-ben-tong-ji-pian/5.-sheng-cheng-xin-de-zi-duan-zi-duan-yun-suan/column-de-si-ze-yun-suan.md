# Column的四則運算

**數值類型**的字段，可以進行加、減、乘、除的四則運算，運算優先級跟數學運算的優先級是一樣的，即**先乘除後加減**

| 運算符  | 描述   |
| ---- | ---- |
| A+B  | A加B  |
| A-B  | A減B  |
| A\*B | A乘以B |
| A/B  | A除以B |

```sql
select 1+2 ,    
    1-2 ,    
    1*2 ,    
    1/2 ,    
    1+2*2 ,    
    (1+2)*2,    
    (1+2)/(1-2) ;
```
